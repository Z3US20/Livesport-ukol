Dobrý den,

jsem Michal Bábela

a vítám Vás na své webové stránce, kterou jsem vytvořil pro Livesport, jako úkol k pozici FE vývojáře ITS.

Moje dovednosti

HTML

HTML používám už 2 roky.
Začali jsme se ho učit v druhém ročníku střední školy. Ve škole jsme probrali základy a zkoušeli vytvořit jednoduché HTML stránky. Mě to začalo bavit, a proto se ve svém volném čase učím více o HTML

CSS

V CSS jsem začal kódovat druhým pololetím v druhém ročníku.
Učili jsme se opět nějaké základní funkce. Jak už jsem psal, začalo mě to bavit a začal jsem se učit ve svém volném čase.

JavaScript

JavaScript jsme se ve škole neučili. Zatím jsem se všechno naučil sám a kopil jsem si JS kurz, který mě celkem rychle naučil alespoň základy JavaScriptu. Nově zkouším programovat v Reactu. V Reactu jsme zatím zkoušel Routes a Styled Components

Moje záliby a koníčky

Sport (fotbal a plážový fotbal), technologie, programování, počítačové hry...

Momentálně hraju plážový fotbal za českou reprezentaci U21 a velký fotbal hraju za naší vesnici Všeradice.

Programuji a učím se v HTML/CSS/JS, Reactu, Python a zkoušel jsem proprojovat database s web stránkou v NodeJS a MySQL.
